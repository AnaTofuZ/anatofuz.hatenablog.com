SICP読んでいるので。

brewでいれられる。簡単。

```sh
$ brew install mit-scheme
```

`/usr/local/Cellar/mit-scheme/9.2_1` あたりに実体が来て
`/usr/local/bin` に先のbinディレクトリ以下のものが貼れるので

`scheme` と入力すると使える
